# Nova Studio – Design Agency Homepage

A modern and responsive Design Agency Homepage built with **Next.js 16** and **Tailwind CSS** as part of a Next.js Developer Internship assignment.

## Features

- Responsive Navigation Bar
- Hero Section with Call-to-Action
- Services Section with Icons
- Portfolio Grid with Optimized Images
- Contact Form with Success Message
- Responsive Design (Mobile, Tablet & Desktop)
- SEO Metadata using Next.js
- Built with Reusable Components

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Lucide React

## Installation

Clone the repository:

```bash
git clone https://github.com/Pradeepkr16/nova-studio-nextjs.git
```

Navigate to the project folder:

```bash
cd nova-studio-nextjs
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

This project is deployed using **Vercel**.
